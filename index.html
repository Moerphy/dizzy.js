The current version of the Dizzy editor supports only the basic content: images and text.

To add text, just start typing. The text will be put on the spot you are currently viewing.
To add images, drag and drop them from your desktop. This works in the latest Chrome and Firefox, not in Opera. It may work in some other browsers, but I didn't have the time to test that yet.


After you put your content on the canvas, switch to the path-mode by clicking on the second button on the toolbar on your left.
In path-mode, you can set the order in which you want to show the things on the canvas. You can also drag and drop the pathnumbers to reorder your path.


Once you have all your content put on the canvas and set the order you should switch to the presentation-mode by clicking "Present" in the menubar on the top. This will hide the toolbar and show buttons for moving back and forth in your path.


Have fun playing with this.
Dizzy is a web based presentation application, heavily inspired by Prezi, that uses a single canvas instead of multiple slides for a presentation.

Unlike Prezi, presentations with Dizzy are made entirely without Flash, using SVG and Javascript only. Also everything is open source under the MIT license.

Sourcecode for the core and the editor are available on Github.

Please keep in mind: this project is still in it's early beta, so there are many open bugs and quirks and a lot of features yet to be implemented.
Would be awesome if you report anything that does not work or behaves strange in the issue tracker
