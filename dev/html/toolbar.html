    • About...
      Dizzy is a web based presentation application, heavily inspired by Prezi, that uses a single canvas instead of multiple slides for a presentation.

      Unlike Prezi, presentations with Dizzy are made entirely without Flash, using SVG and Javascript only. Also everything is open source under the MIT license (code available on Github).

      Please keep in mind that this project is still in it's early beta, so there are many open bugs and quirks and a lot of features yet to be implemented.
      The awesome icons used here are from the fugue icon set by Yusuke Kamiyamane
